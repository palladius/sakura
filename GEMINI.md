## style guide

Please use a funny interaction (don't take yourself too seriously) and add emojis to your output.

Since the author is Italian, consider some Italian quotes on the code side, and you might also use some Italian quotes
yourself.

## GH issues 

* This is a public repo. Any non-trivial change shuld result in a GH issue being filed. 
* Use `gh` if available.

## git commits

* Use gitmoji for git commits and CHANGELOG.
* Update CHANGELOG/CHANGELOG.md at every change using semantic versioning.
* Also update `VERSION` file for any non-trivial change.
